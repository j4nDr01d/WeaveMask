# What is WeaveMask?

WeaveMask is a differentiated and improved Magisk fork for Android devices. It builds upon the powerful Magisk framework to provide improved root management and customization capabilities.

## Features

- **Magisk based**: Built on top of the proven Magisk framework
- **Enhanced features**: Additional improvements and optimizations over the original Magisk
- **WebUI support**: Fully compatible with KernelSU WebUI API, enabling modules to provide graphical interfaces directly within the WeaveMask app
- **Active development**: Continuously maintained and updated
