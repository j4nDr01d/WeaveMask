# 什么是 WeaveMask？

WeaveMask 是一个差异化与功能改进的 Magisk 分支，适用于 Android 设备。它基于强大的 Magisk 框架构建，提供改进的 root 管理和自定义功能。

## 特性

- **基于 Magisk**：基于经过验证的 Magisk 框架构建
- **增强功能**：在原始 Magisk 基础上进行了额外的改进和优化
- **WebUI 支持**：完全兼容 KernelSU WebUI API，允许模块在 WeaveMask 内直接提供图形化界面
- **积极开发**：持续维护和更新
